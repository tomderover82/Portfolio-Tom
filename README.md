# Portfolio-Tom

Personal portfolio website showcasing work experience, technical skills, and background.

Built with Jekyll and hosted on GitHub Pages.

## Setup

1. Clone this repo
2. Install Jekyll: `gem install jekyll bundler`
3. Run locally: `jekyll serve`
4. Visit: `http://localhost:4000/Portfolio-Tom/`

## Customization

Edit these files:
- `index.md` - Homepage
- `about.md` - About section
- `experience.md` - Work history
- `skills.md` - Technical skills
- `contact.md` - Contact info
